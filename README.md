# vanilla-form

## Server

### 1. Install Node Server

```sh
cd server
yarn
```

### 2. Run Node Server

```sh
yarn start
```

The node server will run on localhost:8081

## Client

### 1. Install

```sh
yarn
```

### 2. Run

```sh
yarn start
```

The application will run on development mode on localhost:8080

## Test

Running the tests:

```sh
yarn test
```
